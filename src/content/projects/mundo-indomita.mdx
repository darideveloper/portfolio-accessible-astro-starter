## Descripción del Proyecto

**Mundo Indómita** es un proyecto web _multi-página_ desarrollado para una empresa que ofrece **viajes en grupo para mujeres**. El sitio no es solo una _landing page_, sino una **plataforma completa** diseñada para **informar detalladamente sobre los destinos**, resolver dudas y facilitar el proceso de reserva. La estructura del sitio está pensada para **guiar al usuario** desde el descubrimiento de los viajes hasta la toma de decisión, proporcionando toda la información necesaria para una **experiencia de usuario fluida y segura**. Este proyecto demuestra nuestra capacidad para crear **sitios web de nicho complejos y funcionales**, con múltiples páginas dedicadas a la presentación detallada de un producto o servicio.

- **Página de Inicio**: Sirve como punto de entrada, presentando el concepto, la fundadora, los próximos destinos y testimonios.
- **Página de Destinos**: Una página dedicada a mostrar la lista completa de los **viajes disponibles**, actuando como un índice.
- **Páginas de Destinos Individuales**: Cada viaje cuenta con una página específica (**_Marruecos, Egipto, Gambia, Etiopía, India, Jordania, Uzbekistán_**) para **detallar la información** del itinerario, precios, fechas y otros aspectos relevantes.
- **Página "Acerca de"**: Proporciona información más detallada sobre la fundadora y la filosofía de la empresa.
- **Página de Faqs**: Resuelve las **preguntas más frecuentes** de los clientes potenciales, lo que ayuda a reducir la fricción en el proceso de reserva.
- **Página de Contacto**: Permite a los usuarios **comunicarse directamente** con el equipo de Mundo Indómita.
- **Páginas Legales**: Incluye páginas esenciales como **Aviso Legal**, **Condiciones Generales de Contratación** y **Política de Privacidad** para generar **confianza y transparencia**.

## Tecnologías Utilizadas

- **WordPress:** CMS para la gestión de contenido.
- **Divi:** Constructor de páginas para diseño visual.
- **WPForms:** Plugin para la creación de formularios de contacto.
- **Divi Torque:** Plugin utilizado para crear sliders.
- **Buttonizer:** Plugin para crear botones flotantes y acciones rápidas.

## Retos y Aprendizajes

- **Gestión de contenido de múltiples páginas:** El desafío fue organizar y estructurar un gran volumen de información (15 páginas en total) de manera **coherente y lógica**, asegurando una navegación intuitiva para el usuario.
- **Implementación de un sistema de viajes:** Utilizar un plugin como **_WP Travel Engine_** para gestionar los itinerarios, precios y fechas de manera **eficiente y escalable**.
- **Creación de una experiencia completa:** No solo se trató de un diseño atractivo, sino de construir una **experiencia completa** que respondiera a las necesidades del usuario en cada etapa de su proceso de decisión, desde la inspiración hasta la compra.
- **Integración de elementos legales:** Incluir páginas legales para **proteger tanto al cliente como a la empresa**, lo cual es crucial en la industria del turismo.

## Resultados

El sitio web de Mundo Indómita es una **plataforma robusta y bien organizada** que va más allá de un simple diseño, ofreciendo una experiencia completa al usuario. La estructura _multi-página_ y la integración de plugins especializados permiten a la empresa **gestionar sus ofertas de viaje de forma profesional**, mientras que las páginas de faqs y contacto generan **confianza y facilitan la conversión**.
