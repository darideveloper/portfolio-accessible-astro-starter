## Descripción del Proyecto

**Gana en tu hogar by Coca-Cola®** es una landing page corporativa de alto impacto desarrollada para una de las marcas más icónicas del mundo. El proyecto consistió en crear una página de captación de embajadores para el programa de ventas por recomendación de Coca-Cola.

El desarrollo se centró en dos pilares fundamentales: la **fidelidad absoluta al diseño UI/UX** proporcionado por el cliente y la implementación de **estándares de seguridad empresariales**, garantizando una navegación segura y profesional para los usuarios.

### Funcionalidades y Características Relevantes

- **Réplica Exacta de UI/UX**: Se realizó un trabajo de maquetación meticuloso para asegurar que cada píxel, tipografía y espacio coincidiera con los prototipos de diseño originales.
- **Optimización para Conversión (CRO)**: Estructura clara con botones de llamada a la acción (CTA) estratégicos para maximizar el registro de nuevos embajadores.
- **Infraestructura Empresarial**: Configurada como una página estática bajo las mejores prácticas de seguridad web para mitigar vulnerabilidades en un entorno corporativo.
- **Mapa de Cobertura Geográfica**: Visualización clara que muestra las ciudades con cobertura del programa (CDMX, Puebla, Querétaro, Toluca).
- **Testimonios Reales**: Sección dedicada a la prueba social, utilizando experiencias de embajadores actuales para generar confianza.
- **Diseño Mobile-First**: Garantiza una experiencia fluida desde smartphones, el principal canal de acceso para los potenciales colaboradores.

## Tecnologías Utilizadas

### Framework y Frontend

- **Next.js**: Elegido por su velocidad, SEO y capacidades de generación estática.
- **React**: Para una interfaz dinámica y componentes altamente reutilizables.
- **CSS Avanzado**: Implementación de un sistema de estilos robusto para cumplir con las guías de estilo de marca.

### Seguridad

- **Security Best Practices**: Configuración estricta de headers, validaciones de formularios y manejo seguro de datos estáticos para cumplir con los estándares corporativos.

## Retos y Aprendizajes

- **Precisión Visual**: El mayor reto fue la replicación exacta de un diseño UI/UX muy detallado, lo que nos llevó a perfeccionar nuestras técnicas de maquetación avanzada.
- **Cumplimiento de Estándares**: Trabajar en un proyecto bajo el nombre de Coca-Cola® implicó seguir rigurosos procesos de revisión de seguridad y calidad técnica.

## Resultados

El resultado final es una landing page profesional, rápida y segura que sirve como la puerta de enlace oficial para nuevos embajadores, reflejando fielmente los valores de innovación y cercanía de la marca Coca-Cola®.
